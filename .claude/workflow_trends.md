---
name: 출제 경향 처리 워크플로우
description: 유저가 최근 출제 경향을 주면 trends 폴더에 저장하고, 문제 출제 시 최신 파일만 참고하는 규칙
type: feedback
---

1. 유저가 최근 출제 경향 정보를 주면 `.github/trends/` 폴더에 오늘 날짜(`YYYY_MM_DD.md`)로 파일을 생성하고 내용을 정리한다.
2. 문제를 출제할 때는 `.github/trends/` 폴더에서 가장 최신 파일 하나만 읽어서 해당 내용 기반으로 문제를 제시한다.

**Why:** 출제 경향이 시험마다 바뀌므로 최신 정보만 반영해야 실전에 유용하다.
**How to apply:** 출제 경향 입력 시 → trends 폴더에 날짜 파일 생성. 문제 출제 시 → trends 폴더에서 최신 파일 1개만 참조.
